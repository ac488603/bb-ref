Control Button

down s
attack z

The controls for player one and two are configurable. If you would like to register new keys for those players, you are able to in the menu. Simply navigate to the controls menu under the options menu.


You can also try out our experimental controller feature. Simply navigate to http://bullet-battle.herokuapp.com/ on any mobile device.

You will be presented with a form that requests a session code that can be found in the top right corner of the start screen. Once the server receives you session code, the server returns a controller map that looks like this:


Red is attack and blue is jump. (supports 2-4 players)
